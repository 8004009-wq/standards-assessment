# 更新说明

## 2026-02-27 更新

### ✅ 标准号和评估选项修正

#### 标准号更新

- **原标准号**: GB/T 22239-2019
- **新标准号**: **GA/T 2380-2026**
- **标准名称**: 信息安全技术 网络安全等级保护数据安全基本要求
- **标准类型**: 公共安全行业标准（GA/T）

#### 评估选项更新

根据标准要求，评估选项已更新为：

| 选项 | 说明 | 得分比例 |
|------|------|---------|
| **满足** | 完全满足要求，有完整证据 | 100% |
| **部分满足** | 部分满足要求，有改进空间 | 50% |
| **不满足** | 不满足要求或缺失 | 0% |
| **不适用** | 该项不适用于当前系统 | 不计分 |

**原选项**：符合、部分符合、不符合  
**新选项**：满足、部分满足、不满足

### 🏗️ 7 个评估维度

1. **数据分类分级** (15%) - 5 项
2. **数据采集安全** (15%) - 6 项
3. **数据传输安全** (15%) - 6 项
4. **数据存储安全** (15%) - 7 项
5. **数据使用安全** (15%) - 8 项
6. **数据交换共享** (15%) - 7 项
7. **数据销毁安全** (10%) - 6 项

---

### 🎯 三级系统判定

符合以下条件之一的系统应定为三级：

- 涉及国家安全、国计民生的信息系统
- 涉及重要敏感信息的信息系统
- 用户数量超过 100 万的互联网平台
- 日处理个人信息超过 100 万条
- 存储重要数据超过 100 万条

### 🚀 如何使用

1. **启动系统**
   ```bash
   cd standards-assessment
   ./start.sh
   ```

2. **访问界面**
   - 前端：http://localhost:8080
   - API: http://localhost:8001

3. **创建评估任务**
   - 选择"信息安全技术 网络安全等级保护数据安全基本要求（第三级）"
   - 逐项评估，选择：满足、部分满足、不满足、不适用

4. **查看结果**
   - 总体满足率目标：≥90%
   - 关键项必须 100% 满足
