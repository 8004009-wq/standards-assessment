## ⚠️ 注意事项

### cpolar 免费版限制

- **域名随机**：每次重启 cpolar 会变化
- **带宽限制**：免费版带宽有限
- **会话限制**：只能运行 1 个隧道
- **在线时长**：可能需要定期重启

### 保持服务运行

cpolar 进程已在后台运行，但如果：
- 服务器重启
- cpolar 进程被终止
- 需要更换域名

需要重新启动 cpolar：

```bash
cpolar http 8080
```

### 固定域名（可选）

如果需要固定域名，可以：
1. 升级 cpolar 付费版
2. 或使用自己的服务器 + 域名

## 🆘 故障排查

### 无法访问公网地址

1. **检查 cpolar 是否运行**
   ```bash
   ps aux | grep cpolar
   ```

2. **检查本地服务是否正常**
   ```bash
   curl http://127.0.0.1:8080
   ```

3. **重启 cpolar**
   ```bash
   pkill -9 cpolar
   cpolar http 8080
   ```

### 二维码无法扫描

1. 确认二维码图片完整
2. 尝试直接发送链接
3. 检查网络连接

### 访问速度慢

- cpolar 免费版带宽有限
- 建议升级到付费版
- 或使用服务器公网 IP 方案
